### 运行须知
 - 运行前需要准备的东西：
   + node.js环境（npm包管理器） 
   + vue-cli 脚手架构建工具

+ 进入项目所在文件目录，安装依赖：

  ```
  npm install
  ```

- 运行项目：

  ```
  npm run dev
  ```

- 项目运行在本地默认端口，在浏览器中访问` http://localhost:8080/`，进入项目首页
